# Source Code Folder

This folder contains all Python modules used by the Energy Dashboard project.

Modules:
- `ingest.py` → Reads and validates CSV files
- `aggregates.py` → Calculates daily, weekly, and building-wise summaries
- `models.py` → Contains the BuildingManager class for analysis
- `viz.py` → Plotting functions for visualizing energy usage

These modules are imported into the Jupyter Notebook for processing and analysis.
